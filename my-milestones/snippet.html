The Anatomy of the HTML Tag

Want to search the web? Try Google!

In this snippet of code, there are two tags. A paragraph with its needed beginning and end tag, and an anchor tag with its beginning and end tag.

More info on the Paragraph tag: https://developer.mozilla.org/en-US/docs/Web/HTML/Element/p More info on the Anchor tag: https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a

  1. The anchor tag is the child of the paragraph tag and the paragraph tag is the parent of the anchor tag.
  2. The beginning and end tags for the paragraph tag are siblings. This is the same for the beginning and end tags for the anchor tag.
  3. The names of the attributes of the anchor tag are "href", "id", "class", and "target".
    • The value of the href attribute is the link to Google.com. The href attribute embeds a hyperlink into a HTML page.
    • The value of the id attribute is the-best-link. The id attribute is similar to a class attribute, but can only be used once to identify a specific element.
    • The value of the class attribute is banana. The class attribute gives a general identifier to an element and can be used multiple times.
    • The value of the target attribute is Google. The target attribute specifically tells the computer where to place the link in a sentence.
